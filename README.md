# 人机协同内容助手

基于 Vue 3 的智能内容生成工具,支持小红书和公众号风格文案创作。

## ✨ 功能特性

- 🎨 双风格内容生成 (小红书/公众号)
- 🤖 DeepSeek AI 驱动
- ⚙️ 可自定义提示词
- 💾 配置本地持久化
- 🎯 现代化紫色渐变 UI

## 🚀 在线体验

访问: [https://你的用户名.github.io/仓库名](https://你的用户名.github.io/仓库名)

## 📖 使用说明

1. 选择内容类型 (小红书/公众号)
2. 输入主题内容
3. 点击生成按钮
4. 获得 AI 生成的内容

## ⚙️ 配置

点击配置按钮可以:
- 修改 API Key
- 切换 AI 模型
- 自定义提示词

## 🔒 隐私说明

- 所有配置保存在浏览器本地
- 不会上传任何数据
- API Key 仅用于调用 AI 服务
